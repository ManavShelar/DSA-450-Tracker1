## Run Locally 💻

```
> Clone the repo
    >> For Windows: Git Bash
    >> For Linux: Terminal
    >> git clone https://github.com/ManavShelar/DSA-450-Tracker1
    
> Install the dependencies
    >> yarn install
    
> Start the server
    >> yarn dev
    
> dsa 450 tracker runs on port 3000 of your Local Machine
```
